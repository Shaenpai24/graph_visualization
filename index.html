üå≥ 4. Minimum Spanning Trees (MST)

MST Problem: Find a subgraph that connects all vertices with minimum total edge weight, containing no cycles. Two main algorithms: Prim's (grows from vertex) and Kruskal's (sorts edges).
Prim's Algorithm
Grows tree by adding minimum weight edge to unvisited vertex
Time: O((V+E) log V) | Space: O(V)

Kruskal's Algorithm
Sorts edges, adds them if they don't create cycles
